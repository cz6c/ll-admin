# 常见问题

## 1. 为什么后端启动成功，但前端登录失败

常见原因：

- 前端 `VITE_BASE_URL` 配错了
- 前端请求没打到本地后端
- 数据库初始化数据不完整
- Redis 没启动，导致验证码或登录态异常

## 2. 为什么验证码不显示

优先检查：

- 后端是否启动
- `/captchaImage` 接口是否可访问
- Redis 是否可用
- 是否被浏览器跨域或代理配置影响

## 3. 为什么登录成功后又回到登录页

优先检查：

- token 是否真的保存成功
- 获取用户信息接口是否成功
- 菜单接口是否成功
- token 刷新逻辑是否异常

## 4. 为什么左侧菜单不显示

常见原因：

- `/getRouters` 接口没有返回菜单
- 当前用户角色没有对应菜单权限
- 菜单 `component` 路径配置不正确
- 前端动态路由没有成功注入

## 5. 为什么页面文件已经写好了，但访问不到

重点检查：

- 后端菜单里配置的 `component` 字符串
- `views` 目录中的真实文件路径
- 两者是否完全匹配

## 6. 为什么 Swagger 打不开

优先检查：

- 后端是否真的启动完成
- 当前开发环境 `app.prefix` 是什么
- 浏览器访问路径是否正确
- 是否有端口冲突

## 7. 为什么数据库连不上

检查：

- `src/config/dev.yml` 的 MySQL 配置
- 数据库服务是否启动
- 用户名密码是否正确
- 目标库是否已创建

## 8. 为什么 Redis 连不上

检查：

- `src/config/dev.yml` 的 Redis 配置
- Redis 服务是否启动
- 密码与端口是否正确

## 9. 为什么上传失败

优先检查：

- 文件大小是否超限
- 上传目录是否有权限
- 本地存储与云存储配置是否一致
- 返回的 URL 是否可访问

## 10. 文档项目怎么运行

在 `apps/docs` 执行：

```bash
pnpm install
pnpm dev
```

构建：

```bash
pnpm build
pnpm preview
```
